# Q3D 形象造梦机

## 功能

帮助用户通过上传照片，生成专属 Q 版 3D 虚拟形象，并支持领养为桌面宠物进行 AI 对话和养成互动。支持 9 种骨骼动画（7 种心情 + 攀爬 + 倒挂）、200 级养成系统、Dream Lattice 生成艺术。

## 触发时机

- 用户说"我想做个Q版形象"
- 用户说"生成我的头像"
- 用户说"上传照片变卡通"
- 用户说"创建虚拟角色"
- 用户提到"Q版3D"、"形象造梦机"、"桌面宠物"
- 用户说"宠物养成"、"喂食玩耍"
- 用户说"生成艺术"、"粒子效果"
- 用户说"我的作品"、"作品画廊"

## 输入

- `imagePath`: 用户上传的照片文件路径（绝对路径）
- `style`: 风格选择，可选 `kawaii`（软萌大头）/`guofeng`（国风Q版）/`trendy`（潮玩手办）/`simple`（简约卡通），默认 `kawaii`

## 输出

- `avatarPath`: 生成的 Q 版形象图片路径
- `style`: 实际应用的风格
- `metadataPath`: 生成元数据文件路径
- `sessionId`: 会话 ID（用于后续操作）

## 执行步骤

1. **确认意图**：向用户打招呼，说明可以生成 Q 版形象，请求上传照片。
2. **收集风格**：询问用户想要哪种风格，提供 4 个选项。如果用户没有明确偏好，默认使用 `kawaii`。
3. **调用生成**：
   - 先调用 `q3d_upload_photo` 保存用户上传的照片
   - 再调用 `q3d_generate_avatar` 生成 Q 版形象（TRAE Native 模式下使用 GenerateImage 工具生成后用 `q3d_save_avatar` 保存）
4. **展示结果**：向用户展示生成的图片路径，询问是否满意，提供后续选项。

## 后续操作选项

生成形象后，可根据用户需求提供以下操作：

### 🎨 形象相关
- **重新生成**：调用 `q3d_regenerate_avatar` 换种子/换风格
- **换风格**：4 种风格任选，一键切换
- **查看作品**：调用 `q3d_manage_gallery` 查看所有作品

### 🦴 3D 预览
- **骨骼动画预览**：调用 `q3d_create_bones_preview` 查看 9 种动画
  - 7 种心情：平静/开心/兴奋/困倦/好奇/难过/喜爱
  - 2 种行为：攀爬（借鉴 Shimeji）、倒挂爬行
- **心情切换**：调用 `q3d_control_mood` 切换不同心情动画

### 🐾 桌面宠物
- **领养宠物**：调用 `q3d_spawn_pet` 创建桌面宠物页面
- **AI 对话**：调用 `q3d_chat_with_pet` 与宠物聊天
- **宠物养成**：调用 `q3d_pet_care` 进行喂食、玩耍、改名等（200 级好感度系统）
  - 喂食：小食/正餐/甜点
  - 玩耍：玩球/追逐/解谜
  - 性格：友好/调皮/慵懒/害羞
- **心情控制**：调用 `q3d_control_mood` 设置宠物心情

### ✨ 生成艺术
- **Dream Lattice**：调用 `q3d_generate_dream_lattice` 生成造梦晶格粒子艺术
  - 4 种风格色板：软萌粉/国风沙/潮玩霓虹/简约灰
  - 可调粒子数量和动画速度

### 📊 信息查询
- **项目信息**：调用 `q3d_get_project_info` 查看功能和配置

## 心情动画列表

| 动画 ID | 名称 | 类型 | 描述 |
|---------|------|------|------|
| idle | 平静 | 心情 | 缓慢呼吸 + 轻微摇摆 |
| happy | 开心 | 心情 | 弹跳 + 手臂挥动 |
| excited | 兴奋 | 心情 | 跳跃 + 全身缩放 + 高举双手 |
| sleeping | 困倦 | 心情 | 慢速呼吸 + 低头 + 放松 |
| curious | 好奇 | 心情 | 歪头 + 前倾 + 手臂微抬 |
| sad | 难过 | 心情 | 下垂 + 颤抖 + 低头含胸 |
| love | 喜爱 | 心情 | 心跳脉冲 + 双手抱胸 |
| climbing | 攀爬 | 行为 | 向上攀爬姿态（借鉴 Shimeji 经典行为） |
| crawling_upside | 倒挂 | 行为 | 顶部倒挂爬行（借鉴 WindowPet） |

## 宠物养成系统

- **等级系统**：200 级，经验值随等级递增
- **五维属性**：好感度、饱食度、精力值、清洁度、快乐值
- **四种性格**：友好、调皮、慵懒、害羞
- **互动方式**：喂食、玩耍、改名、设置性格
- **状态持久化**：自动保存到 pet-status.json

## 失败策略

- **API 未配置**：如果 `q3d_generate_avatar` 返回 API 未配置错误，提示用户：
  "TRAE Native 模式已启用，可以直接使用 TRAE 的图像生成能力。让我为你生成形象~"
  （TRAE Native 模式下使用 GenerateImage 工具 + q3d_save_avatar）
- **API 调用失败**：如果返回 API 错误（超时、余额不足、内容审核拒绝），自动降级到下一个 Provider（TRAE Native → External API → Mock），并向用户说明情况。
- **图片过大**：如果上传图片超过 4MB，建议用户压缩后再上传。
- **生成不满意**：提供重新生成（换种子值）或切换风格选项。

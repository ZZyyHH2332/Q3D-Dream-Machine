# Q3D 形象造梦机

> 图片一键生成 Q 版 3D 虚拟角色工具 —— 生活娱乐赛道参赛作品

## 项目简介

Q3D 形象造梦机是一款面向普通用户的轻量化 AI 虚拟形象生成工具。上传照片或手绘素材，AI 自动提取五官、外形、穿搭特征，数十秒内生成专属 Q 版 3D 角色模型与配套 2D 角色插画，同时生成可交互的桌面宠物进行 AI 虚拟角色对话。

## 核心功能

| 模块 | 功能说明 |
|---|---|
| **图片上传** | 支持本地上传、拖拽上传、快捷素材（自拍/手绘/宠物） |
| **风格切换** | 软萌大头、国风 Q 版、潮玩、简约卡通 4 种风格 |
| **3D 预览** | CSS 3D 立方体，支持鼠标/触摸拖拽旋转 |
| **Mermaid 流程图** | 完整展示 AI 生成链路 |
| **桌面宠物** | 生成后宠物出现，可拖拽、双击互动、右键菜单 |
| **PiP 置顶** | Chrome/Edge 支持 Document Picture-in-Picture 真正置顶 |
| **AI 对话** | 模拟 15+ 组问答，打字机效果，支持语音输入 |
| **TRAE 唤起** | `solo-cn://` 深度链接一键唤起 TRAE IDE |

## 技术栈

- **前端**：原生 HTML5 / CSS3 / JavaScript（零框架依赖）
- **3D 渲染**：CSS 3D Transforms
- **流程图**：Mermaid.js（CDN）
- **字体**：Google Fonts（ZCOOL KuaiLe / Noto Sans SC）
- **语音**：Web Speech API
- **PiP**：Document Picture-in-Picture API

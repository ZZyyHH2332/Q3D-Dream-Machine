# Q3D 形象造梦机

## 功能
帮助用户通过上传照片，生成专属 Q 版 3D 虚拟形象，并支持领养为桌面宠物进行 AI 对话。

## 触发时机
- 用户说"我想做个Q版形象"
- 用户说"生成我的头像"
- 用户说"上传照片变卡通"
- 用户说"创建虚拟角色"
- 用户提到"Q版3D"、"形象造梦机"、"桌面宠物"

## 输入
- `imagePath`: 用户上传的照片文件路径（绝对路径）
- `style`: 风格选择，可选 `kawaii`（软萌大头）/`guofeng`（国风Q版）/`trendy`（潮玩手办）/`simple`（简约卡通），默认 `kawaii`

## 输出
- `avatarPath`: 生成的 Q 版形象图片路径
- `style`: 实际应用的风格
- `metadataPath`: 生成元数据文件路径

## 执行步骤

1. **确认意图**：向用户打招呼，说明可以生成 Q 版形象，请求上传照片。
2. **收集风格**：询问用户想要哪种风格，提供 4 个选项。如果用户没有明确偏好，默认使用 `kawaii`。
3. **调用生成**：
   - 先调用 `q3d_upload_photo` 保存用户上传的照片
   - 再调用 `q3d_generate_avatar` 生成 Q 版形象
4. **展示结果**：向用户展示生成的图片路径，询问是否满意，提供"重新生成"、"换风格"、"生成3D预览"、"领养桌面宠物"、"查看作品页"等后续选项。所有作品会自动收录到作品展示页，打开项目根目录的 `index.html` 即可查看你的作品画廊。

## 失败策略

- **API 未配置**：如果 `q3d_generate_avatar` 返回 API 未配置错误，提示用户：
  "需要先配置 AI 图像生成 API。请复制 `mcp-server/.env.example` 为 `.env`，填写你的 API Key 和 API 基地址。"
- **API 调用失败**：如果返回 API 错误（超时、余额不足、内容审核拒绝），显示具体错误信息和建议：
  - 超时：请检查网络连接或稍后重试
  - 余额不足：请充值 API 账户
  - 内容审核：请尝试其他照片
- **图片过大**：如果上传图片超过 4MB，建议用户压缩后再上传。
- **生成不满意**：提供重新生成（换种子值）或切换风格选项。
